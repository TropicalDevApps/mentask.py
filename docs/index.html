INTERFACE_REF

Designed for the terminal. mentask provides a high-performance, color-coded TUI with real-time streaming.

COMMAND_ID DESCRIPTION
/help Manifests current ToolRegistry and configuration state.
/model <id> Hot-swaps LLM provider and re-initializes buffers.
/mode [auto|manual] Toggles the TrustManager confirmation bypass.
/trust <path> Authorizes a directory for recursive file operations.
/init Bootstraps .mentask/ with SQLite and identity.
/artifacts Introspects the forge/ incubator for generated tools.
/undo Performs a state rollback of the last file mutation.
/usage Calculates real-time token consumption and USD cost.
/stats Session summary (messages, tools, files edited).
